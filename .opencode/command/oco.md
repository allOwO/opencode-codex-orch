# OCO

`OCO` is the short name for `opencode-codex-orch`.

`opencode-codex-orch` is a lean OpenCode plugin fork focused on multi-agent orchestration, Codex-style prompting, and a smaller, more practical toolchain.

## Highlights

- Lean fork of `oh-my-openagent` with a rewritten Sisyphus prompt
- Multi-agent workflow with Sisyphus, Oracle, Librarian, Explore, Metis, and Momus
- Built-in LSP tools, Context7 docs lookup, Exa web search, and grep.app code search
- Bun-first TypeScript codebase with plugin hooks, slash commands, skills, and MCP integration
